Angular c'est quoi ?

Un framework

Model-View-Whatever

 

Model : des POJO (Plain Old Javascript Objects)

View : le DOM (Document Objet Model)

Whatever : ce que vous voulez

Que fourni Angular ?

  • Injection de dépendances
  • Modularisation du code
  • Templates
  • 2-ways databinding
  • Navigation (Routing)
  • Séparation des préoccupations (SoC)
  • Tests unitaires (via Jasmine)
  • Tests e2e (via protractor)
La liste n; est pas exhaustive, il manque par exemple: . les animations . securite . l'internationalisation . les validations de formulaires ...
