Les filtres

=

Formattage

Cas d'utilisations

  • Formatter des valeurs (ex dates, monnaies)
  • Trier des collections
  • Filtrer des collections
  • ...

Syntaxe


    {{ expression | filter }}

Ex: 


    {{ 'hello' | uppercase }}  # => HELLO

les filtres peuvent être chaînés


    {{ expression | filter | filter }}

les filtres peuvent avoir des arguments


    {{ expression | filter:argument1:argument2:... }}

Creer ses filtres


var myApp = angular.module('myApp', []);

myApp.filter('stars', function() {
  // retourne la function de filtrage
  return function(input) {
    if (input === undefined) { input = ''; };
    return '*** ' + input + ' ***';
  }
});
  

     {{ 'hello' | stars }}  =>  *** hello ***
