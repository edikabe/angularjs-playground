Single Page Applications (SPA)

(Applications monopages)

Terme introduit par Steve Yen en 2005

“ A single-page application (SPA), is a web application or web site that fits on a single web page with the goal of providing a more fluid user experience akin to a desktop application. ”

Les buts des SPA

  • Rendre plus fluides les applications web
  • Fournir une expérience utilisateur proche d'une application native

Les techniques utilisées

  • Chargements asynchrones (AJAX, Websockets, ...)
  • Modifications dynamiques du DOM

Les problèmes induits par les SPAs

  • Gestion de l'historique
  • Bookmarks
  • Référencement dans les moteurs de recherche
  • Memory leaks
  • ...
